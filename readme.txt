== Description ==
Insere um icone com link direto para o seu app de mensagens de forma fácil e rápida no seu site. Ideal para solicitar orçamento direto do site.

Possibilitando também monitorar a quantidade de cliques no Google Analitycs. Além de poder mudar a posição do icone esquerda e direita e na vertical.
Basta colocar seu telefone, mensagem e começar a usar.

== Installation ==

1. Envie o plugin para a pasta `/wp-content/plugins/` via ftp ou pelo repositório do Wordpress.
2. Ative pela área administrativa do WordPress.
3. Coloque seu telefone conforme indicado no campo, escreva uma mensagem e pronto.
4. Acompanhe quantos cliques foram dados pela sua conta do Google Analitycs.


== Frequently Asked Questions ==

**Funciona em telefones?**
Sim, o plugin diferencia o acesso mobile dos desktops.


**É preciso configurar o Analitycs**
Não, os paramentros são passados caso seu código esteja instaldo.
